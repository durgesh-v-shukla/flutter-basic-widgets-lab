# flutter-basic-widgets-lab

A structured collection of 50 Flutter widget mini apps.
Each widget is implemented as a small standalone example to demonstrate its usage, layout behavior, and interaction in Flutter UI development.

## Project Goals
- Learn fundamental Flutter widgets
- Understand layout systems
- Practice UI composition
- Provide simple runnable widget examples

## Structure
- widgets_examples/: 50 distinct isolated Flutter projects demonstrating exactly one widget each.
- master_widget_app/: A unified Flutter application with a lazy-loaded list providing a menu and documentation of all 50 examples.
- docs/: Comprehensive line-by-line breakdown of every file, layout, and logic block in this repository.
